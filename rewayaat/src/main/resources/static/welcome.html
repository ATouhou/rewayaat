Rewayaat.info

Welcome to the Rewayaat Hadith Database Project. This project has the following aims:

Collect

To collect all the sayings of the Holy Prophet and his family in a common, searchable repository available to everyone.

Contribute

To allow and encourage contributions from the community, thereby improving the quality of translated hadith for everyone.

Analyze

To provide a first class search experience for those wishing to gather deep insights across large sets of data in a short amount of time.

Getting Started

Use the search bar at the top of this page to search for hadith using one of the following methods:

Method 1: Simple Query

Enter relevant search terms, separated by a space. Phrases, or search terms that include a space should be surrounded by quotation marks.

For example, to search for hadith that are from the book Al-Kafi, are about intelligence, and contain the phrase "a group of our associates", one would search:

Al-Kafi intelligence "a group of our associates"

Click here to see this query in action!

Method #2: Elastic Search Query String Query

Rewayaat stores all narrations as documents in an Elastic Search index. If you would like to take advantage of specific search capabilities provided by Elastic Search, you may search using valid Elastic Search v5.3 Query String syntax.

For example, to search for hadith that are from the book Al-Kafi, must include intelligence, must not include ignorance, possibly include العقل and contain a phrase similar to "one of our associates", one can search:

book:Al-Kafi +intelligence -ignorance العقل "a group of our associates"~3

Click here to see this query in action! Refer to the provided documentation link above for more information.
